## Installation

```bash
$ pnpm install
```

## Running the app

```bash
$ pnpm run start:debug
```

## Testing

```bash
$ pnpm run test
```

## Linting

```bash
$ pnpm run lint
```

## Formatting

```bash
$ pnpm run format
```

## Requirements

Node.js >= 18.18.0
pnpm >= 9.4.0
